## Шаг 1: Инфраструктура и архитектура

- Настройка Next.js, Tailwind, Zustand.

- Формирование директорий по методологии FSD.

- Создание файла components.json и базовых типов моковых данных.

## Шаг 2: Базовый UI и Лендинг

- Разработка изолированных компонентов в shared/ui (кнопки, модальное окно).

- Верстка интерактивной сетки лендинга с кастомным курсором.

## Шаг 3: Глобальное состояние и Авторизация

- Подключение Zustand для управления балансом кредитов и статусом входа.

- Создание хедера с логикой заглушки авторизации через Google.

- Смена состояний хедера (кнопка входа -> аватар с балансом).

## Шаг 4: Студия — Режим идеи

- Верстка главного экрана студии с полем ввода идеи.

- Реализация искусственной задержки для имитации генерации стартового сценария.

## Шаг 5: Студия — Режиссура и Результат

- Верстка панели расширенных настроек (персонажи, стиль).

- Создание сетки сториборда.

- Имитация поэтапной генерации элементов (текст -> драфт картинки).

- Верстка вертикального плеера-заглушки (9:16) для финального результата с кнопкой экспорта.

- При выполнении работы использовать скиллы /sasha и /wegpu-threejs для верстки и интерактивных элементов.
