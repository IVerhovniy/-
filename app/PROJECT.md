# AI Video Studio — Документация проекта

## Описание
AI Video Studio — веб-приложение для генерации видео из текстовых идей.
Frontend-only приложение с моковыми данными и имитацией генерации.

## Технологический стек
- **Next.js 15** (App Router, TypeScript)
- **Tailwind CSS v4** (CSS-first конфигурация)
- **Zustand** (управление состоянием)
- **Three.js** (планируется для 3D-элементов лендинга)

## Структура проекта (FSD)
```
src/
├── app/              # Next.js App Router (страницы, layout)
├── shared/           # Переиспользуемые утилиты, типы, конфиг
│   ├── api/          # Моковые данные
│   ├── config/       # Константы приложения
│   ├── lib/          # Утилиты (cn, formatCredits, mockDelay)
│   ├── types/        # Базовые TypeScript-типы
│   └── ui/           # Базовые UI-компоненты (кнопки, модалки)
├── entities/         # Бизнес-сущности
│   ├── project/      # Проект (сценарий, сториборд, видео)
│   └── user/         # Пользователь
├── features/         # Фичи
│   ├── auth/         # Авторизация (заглушка Google OAuth)
│   └── studio/       # Генерация видео
└── widgets/          # Составные виджеты
    ├── header/       # Хедер с навигацией
    ├── landing-hero/  # Главный экран лендинга
    └── studio-workspace/ # Рабочая область студии
```

## Дизайн
- **Фон**: Синий градиент (тёмно-синий → голубой) в стиле Sui
- **Стиль**: Вдохновлён unseen studio — премиальный, минималистичный
- **Glass-морфизм** для карточек и поверхностей
- **Кастомный курсор** (планируется)

## Ограничения (ЗАГЛУШКИ)
- Генерация видео имитируется через setTimeout
- Авторизация — мок без реального OAuth
- Нет бэкенда, все данные локальные
- Нет реального биллинга/списания кредитов

## Запуск
```bash
cd app
npm run dev
```
